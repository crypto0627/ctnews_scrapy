## Starting 
Make sure you have an Anaconda! <br>
Enviroment:windows10/python3.10.5
```bash
# Clone this project
$ https://github.com/crypto0627/ctnews_scrapy.git
$ cd ctnews
$ pip install Scrapy
$ scrapy crawl headlines -o data.csv
```
產生的csv檔即為爬下來的資料，若CSV檔顯示亂碼，只要用記事本開啟並另存新檔為ANSI編碼。
